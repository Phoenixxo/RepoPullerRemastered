# RepoPullerRemastered (RPR)

A terminal UI for inspecting Git repositories with precision and accuracy.

Inspired by `lazygit`, built with Rust, `ratatui`, and `crossterm`

## Quickstart

---
Install and build:

```bash
git clone https://github.com/Phoenixxo/RepoPullerRemastered
cd RepoPullerRemastered
cargo build --release
cargo run
